Compréhension du Projet :

En tant que Data Scientist pour le PSBA, notre mission consiste à évaluer la capacité des entreprises à rembourser les prêts garantis par la SBA. Il est impératif d'éviter tout "data leaking" en ne utilisant que les informations disponibles avant la prise de décision.

Processus du Projet :

Partie 1 : Nettoyage, EDA et Modélisation

  • Nettoyage des Données : Élimination des valeurs manquantes, gestion des outliers, etc.
  • Analyse et Compréhension des Données : Analyse univariée, bivariée, tests statistiques.
  • Préparation des Données pour le Modèle : Pré-traitement, sélection des features, feature engineering.
  • Modélisation : Utilisation du modèle de classification CatBoost avec explication du fonctionnement.

Partie 2 : Mise en Production

  • Utilisation de FastAPI : Rendre le modèle accessible via une API.
  • Développement d'une Interface avec Django : Création d'une application web pour consommer l'API.
  • Composition de l'Application Web : API (FastAPI) pour servir le modèle & interface utilisateur (Django) pour consommer l'API.
